9. 跨团队合作

甲程序员 Fork仓库 –> 甲程序员讲仓库clone到本地进行修改 –> 甲程序员推送到远程 –> 甲程序员发起pull request –> 原作者审核 –> 原作者进行合并代码

10. SSH免登陆

生成秘钥后，打开公钥，复制内容，在远程仓库中，Setting –> SSH设置 –> 添加SSH秘钥 –> 粘贴添加。然后在push的时候通过ssh链接进行推送，即可实现不用账号密码登录
